# Technology Overview Site

A comprehensive, developer-focused learning resource covering key modern technologies with practical code samples.

## Features

- **Clean, minimal design** - Professional appearance optimized for learning
- **Navigable site structure** - Easy browsing between technologies and use cases
- **Code-focused content** - Each use case includes real, practical code samples
- **Quick learning** - Concise overviews and explanations (learn in minutes)
- **Multiple technologies** - Four key platforms covered

## Technologies Covered

### 1. Google ADK (Android Development Kit)
- **Overview**: Native Android development with Kotlin and Jetpack
- **Use Cases**:
  - Consumer Mobile Apps
  - Enterprise Applications
  - Real-Time Communication
  - IoT & Embedded Systems

### 2. CrewAI
- **Overview**: Autonomous AI agent orchestration framework
- **Use Cases**:
  - Research & Analysis
  - Content Generation
  - Software Development
  - Customer Support

### 3. LangChain
- **Overview**: Framework for LLM-powered applications
- **Use Cases**:
  - RAG Systems (Retrieval-Augmented Generation)
  - Conversational AI
  - Task Automation
  - Data Analysis

### 4. GCP Vertex AI
- **Overview**: Enterprise ML platform for building and deploying models
- **Use Cases**:
  - Computer Vision
  - Predictive Analytics
  - NLP & Text Analysis
  - ML Ops & Scaling

## How to Use

1. **Open the site**: Open `index.html` in a web browser
2. **Browse technologies**: Click on any technology card to view overview
3. **Explore use cases**: Each overview page lists typical use cases
4. **Learn details**: Click on a use case to view code samples and explanations
5. **Navigate back**: Use breadcrumb navigation or back links to return
