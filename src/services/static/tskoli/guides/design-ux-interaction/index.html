Micro-interactions are important for different reasons, like giving your project a more natural feel and conveying your message in a more compelling way. In this guide you will explore this topic and learn to create a presentation to demonstrate your ideas.

Knowledge

  • Understand what are micro-interactions

Skills

  • Be able to demonstrate an interaction design plan

Topics

  • Micro-interactions
    • Transitions design
    • Interactions design
  • Design presentation
    • InVision or similar

Action Points

  • Learn about micro-interactions in an UX context
  • Learn how to create a presentation of interactions and transitions on InVision or similar software

Deliverable: Navigable design files (prototype)

  • Submit:
    • InVision Project URL
    • Screencast URL
  • Minimum requirements: 
    • Create a prototype for a given briefing using InVision, containing the transitions and interactions design
    • Create a screencast to present the project, walking the client through the application flow
